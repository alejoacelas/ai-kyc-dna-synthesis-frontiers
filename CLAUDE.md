# Guidance

Use uv for package management and running python scripts.

## Project summary

Replication package for a paper evaluating whether large language models can assist with legitimacy screening of synthetic nucleic acid customers. The study tests five LLMs on five verification tasks against an expert human baseline across flag accuracy, source quality, source fidelity, and cost — finding Gemini 2.5 Pro matches human flag accuracy at ~1/10 the cost.

### Main manuscript files

- `paper/main.tex` — main manuscript (Frontiers template)
- `paper/supplementary.tex` — supplementary material (appendix)

## Repository organization

This is a replication package: anonymized data in, figures and tables out.

### Data

- `data/tests.csv` — one row per test assertion. Key columns: `model_label`, `model_type`, `is_human_baseline_dataset`, `test_category`, `metric_name`, `pass`, `institution_country`, `customer_type`
- `data/responses.csv` — one row per model response. Key columns: `model_label`, `total_cost`, `latency_ms`, `num_web_searches`
- `data/token_pricing.yaml` — per-token pricing for each model
- `data/customers/full_dataset.csv` — 134 anonymized customer profiles
- `data/customers/human_baseline_subset.csv` — 40-profile subset for human baseline comparison
- `data/annotations/ground_truth_flags.json` — manually verified flag determinations
- `data/annotations/blind_gradings.json` — blind re-grading data for LLM-as-judge validation
- `data/annotations/agreements.json` — inter-rater agreement metadata
- `data/annotations/flag_error_comments.json` — human notes on flag accuracy errors

### Scripts

All scripts read from `data/` and output to `paper/figures/`.

- `scripts/style.py` — shared colors, model labels, ordering, matplotlib config
- `scripts/generate_main_figures.py` — Figures 2-5 (main manuscript)
- `scripts/generate_model_performance.py` — Figures E1-E8 (supplementary)
- `scripts/generate_full_dataset.py` — Figures F1-F5 (supplementary)
- `scripts/generate_flag_accuracy.py` — Figures G1-G9 (supplementary)
- `scripts/generate_cost_latency.py` — Figures H1-H7 (supplementary)
- `scripts/generate_source_analysis.py` — Figures I1-I6 (supplementary)
- `scripts/compute_tables.py` — Tables 1-2 and inline statistics N1-N11

### Paper

- `paper/main.tex` — main manuscript (Frontiers template)
- `paper/supplementary.tex` — supplementary material
- `paper/references.bib` — bibliography
- `paper/figures/Figure1.png` — static methodology diagram (committed)
- Other figures in `paper/figures/` are generated by scripts

### Prompts

- `prompts/screening.txt` — main screening prompt
- `prompts/background_work.txt` — background work search prompt
- `prompts/evaluation/*.yaml` — evaluation test definitions

## Reproduction

```bash
uv sync
make figures   # generates all figures in paper/figures/
make tables    # prints table values and inline statistics
```
