# Guidance

Use uv for package management and running python scripts.

## Repository organization

This is a replication package: anonymized data in, figures and tables out.

### Data

- `data/tests.csv` ‚Äî one row per test assertion. Key columns: `model_label`, `model_type`, `is_human_baseline_dataset`, `test_category`, `metric_name`, `pass`, `institution_country`, `customer_type`
- `data/responses.csv` ‚Äî one row per model response. Key columns: `model_label`, `total_cost`, `latency_ms`, `num_web_searches`
- `data/token_pricing.yaml` ‚Äî per-token pricing for each model
- `data/customers/full_dataset.csv` ‚Äî 134 anonymized customer profiles
- `data/customers/human_baseline_subset.csv` ‚Äî 40-profile subset for human baseline comparison
- `data/annotations/ground_truth_flags.json` ‚Äî manually verified flag determinations
- `data/annotations/blind_gradings.json` ‚Äî blind re-grading data for LLM-as-judge validation
- `data/annotations/agreements.json` ‚Äî inter-rater agreement metadata
- `data/annotations/flag_error_comments.json` ‚Äî human notes on flag accuracy errors

### Scripts

All scripts read from `data/` and output to `paper/figures/`.

- `scripts/style.py` ‚Äî shared colors, model labels, ordering, matplotlib config
- `scripts/generate_main_figures.py` ‚Äî Figures 2-5 (main manuscript)
- `scripts/generate_model_performance.py` ‚Äî Figures E1-E8 (supplementary)
- `scripts/generate_full_dataset.py` ‚Äî Figures F1-F5 (supplementary)
- `scripts/generate_flag_accuracy.py` ‚Äî Figures G1-G9 (supplementary)
- `scripts/generate_cost_latency.py` ‚Äî Figures H1-H7 (supplementary)
- `scripts/generate_source_analysis.py` ‚Äî Figures I1-I6 (supplementary)
- `scripts/compute_tables.py` ‚Äî Tables 1-2 and inline statistics N1-N11

### Paper

- `paper/main.tex` ‚Äî main manuscript (Frontiers template)
- `paper/supplementary.tex` ‚Äî supplementary material
- `paper/references.bib` ‚Äî bibliography
- `paper/figures/Figure1.png` ‚Äî static methodology diagram (committed)
- Other figures in `paper/figures/` are generated by scripts

### Prompts

- `prompts/screening.txt` ‚Äî main screening prompt
- `prompts/background_work.txt` ‚Äî background work search prompt
- `prompts/evaluation/*.yaml` ‚Äî evaluation test definitions

## Reproduction

```bash
uv sync
make figures   # generates all figures in paper/figures/
make tables    # prints table values and inline statistics
```
